data_excel.py 에 속해있는 함수 get_data_json은 

주식의 이름을 입력받아서,   해당 주식의 [날짜, 가격] 을 리턴해줍니다.


하지만 웹에서의 전달성을 위해 json으로 리턴값이 나오므로, 

아래 사항을 한 번 읽어보기를 추천합니다.




____________________________________________________________________________________________________






일단 import로 'from data_excel import get_data_json' 을 입력해줍니다!

____________________________________________________________________________________________________

※  get_data_json 함수의 기능

1. 주식의 이름을 string으로 입력받아서 json 형태로 리턴해줍니다.

/// json 형태란?    [javascript 언어]
____________________________________________________________________________________________________

string 객체로 취급받는 하나의 파이썬 type 입니다.

보통 웹 환경에서는 파이썬의 list 등의 type이 전달되기 힘들기 때문에


- 파이썬의 객체 (ex/ list,dict) 를 json으로 인코딩

- json(string) 객체로 웹 환경에서 전달

- 필요한 곳에서 디코딩해서 다시 파이썬 객체 (ex/ list,dict)로 변환!


이러한 방식으로 이용됩니다!

★☆★☆★☆★☆★☆   json 활용법 [중요!!!!!!!!]!]     ☆★☆★☆★☆★☆★

____________________________________________________________________________________________________

get_data_json 함수는 json 형태로 리턴값을 제공합니다!

그렇기 때문에 이 값을 사용하기 위해서는 json 디코딩을 해줘야 하는데요,

간단합니다!

- 먼저 json을 import 해줍니다
(python 2.6이후로 json은 python 기본 패키지 임으로 'import json'만 해주면 됩니다.)

- get_data_json을 통해서 데이터를 받습니다
( ex /  example_json = get_data_json('카카오')     )

- 해당 데이터를 디코딩 해줍니다.
(ex  /  decoded_json = json.loads(example_json)         )

끝!

____________________________________________________________________________________________________

※디코딩된 get_data_json의 리턴 값은

[  [ ' 날짜 ' ,  가격 ] ,    [ ' 날짜 ' ,  가격 ] ,    [ ' 날짜 ' ,  가격 ] , ............. ,  [ ' 날짜 ' ,  가격 ]    ]

과 같은 이중 리스트로 되어있습니다. 

(날짜는 '20190805' 와 같이 'yyyymmdd' 형식(string)으로 되어있습니다)

(가격은 523000 와 같이 기본적인 int 형식으로 되어있습니다.)
